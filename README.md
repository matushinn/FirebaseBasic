# FirebaseBasic

## 何を作るのか？
FireStoreを使ったSNSアプリ

## 何を学べるか？
* Firebaseを使った匿名ログイン
* UserDefaultsを使ったログイン情報の保持
* EMAlertControllerを使ったSNSシェアの方法
* CustomCellの使い方
